# Kitchen Simulator

A place to break a few eggs.

## Dependencies

On Ubuntu:
```
sudo apt-get install build-essential
```

## Build

```
cd kitchen-simulator
make
./bin/omelette
```

## libkitchensink

libkitchensink can be used to write programs where you simulate making food.

libkitchensink
├── stove
├── pan
└── fridge
